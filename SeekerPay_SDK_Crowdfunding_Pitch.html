SeekerPay SDK

Crowdfunding Pitch

SeekerPay SDK is an open-source Flutter SDK for building SKR payment experiences on Solana Seeker. It is designed for developers who want production-ready building blocks for QR payments, NFC tap-to-pay, Bluetooth proximity payments, split bills, domain resolution, and Seeker-native payment flows.

This is not a concept. The SDK is already built from a working SeekerPay product and extracted into reusable packages. Crowdfunding will help complete the next stage: stronger documentation, more sample apps, feature expansion, and ongoing maintenance.

The SeekerPay app is already available on the Solana dApp Store under DeFi & Trading, and this SDK is the open-source developer layer extracted from that real app experience.

Why Back SeekerPay SDK

Most developers do not want to rebuild wallet signing, Solana payment flows, domain resolution, QR handling, or split-bill logic from scratch. SeekerPay SDK reduces that work and gives Flutter developers a faster path to shipping SKR-native apps on Seeker.

By backing this project, supporters help create shared infrastructure for the Seeker ecosystem instead of a single closed product.

What Is Already Built

The SDK already includes working package foundations for:

  • seekerpay_core for RPC access, wallet connection, SKR payments, balances, activity, and pending transactions
  • seekerpay_qr for Solana Pay URL encoding and QR payment flows
  • seekerpay_nfc for NFC payment handoff integration
  • seekerpay_bluetooth for nearby device-to-device payment handoff
  • seekerpay_split for split bills and payment tracking
  • seekerpay_domains for .skr and .sol resolution plus identity checks
  • seekerpay_ui for reusable SeekerPay UI components

Already Created Features

  • SKR payment flow for Flutter apps
  • Wallet signing integration for Seeker-native payment flows
  • QR payment request generation and decoding
  • NFC payment handoff support
  • Bluetooth nearby payment handoff support
  • Split bill creation and payment verification
  • .skr and .sol domain resolution
  • Seeker identity verification support
  • Reusable payment UI components
  • Example apps for key SDK flows

Who This Is For

SeekerPay SDK is for:

  • Flutter developers building on Solana Seeker
  • teams that want SKR payments without rebuilding the full stack
  • merchant and checkout app builders
  • wallet-adjacent apps that need payment requests and transfer flows
  • developers who want open-source Seeker-native payment infrastructure
